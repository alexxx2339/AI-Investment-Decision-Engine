\# AI Investment Decision Engine



This project uses Python and machine learning to analyze NVIDIA historical stock data and estimate next-day stock direction.



\## Project Overview



The project downloads historical NVDA stock data, engineers financial features, trains multiple machine learning models, and compares their performance.



\## Tools Used



\- Python

\- JupyterLab

\- pandas

\- yfinance

\- matplotlib

\- scikit-learn

\- XGBoost



\## Features Created



\- Daily returns

\- 20-day moving average

\- 50-day moving average

\- Bullish/Bearish trend signal

\- Tomorrow Up target variable



\## Machine Learning Models



The project compares:



\- Logistic Regression

\- Random Forest

\- XGBoost



\## Results



| Model | Accuracy |

|---|---:|

| Logistic Regression | 54% |

| Random Forest | 53% |

| XGBoost | 59% |



XGBoost performed best on the test dataset.



\## Conclusion



This project demonstrates an end-to-end machine learning workflow using real financial market data, including data collection, feature engineering, model training, evaluation, and prediction.
